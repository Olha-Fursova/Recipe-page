### Main task

This challenge will help you focus on writing semantic HTML. Ensure you think through what HTML elements are most appropriate for each piece of content.
Your challenge is to build out this recipe page and get it looking as close to the design as possible.

You can use any tools you like to help you complete the challenge. So if you've got something you'd like to practice, feel free to give it a go.

### Built With

- HTML5 mark-up
- CSS3 styles
- Flexbox

## Conclusions

The component itself is very easy to build, there were no difficulties in creating styles in general, but the main challenge was building the markup according to semantics. Using appropriate tags for specific elements is a very important task, since the result significantly affects the ease of navigating the page and high-quality cooperation with search engines.

### What have I achieved

Working with this kind of markup can be tedious and requires a meticulous approach. For me, one of the achievements was to build stylized proportional lists and tables, since before that I usually did not have practice with styling all their elements, and could mostly avoid them by magically removing element markers. So this time I had practice with this task and was able to remember the important components in building these elements.
